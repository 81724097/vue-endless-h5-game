# Endless

> 一个用 Vue.js 开发的地穴冒险类游戏;

![image](https://github.com/bastarder/Endless/blob/master/game.gif)

## 项目安装&启动

``` bash
# Step 1
npm install

# Step 2
npm run dev

# build
npm run build

```

## 可能发生的问题
1. 如果出现 `webpack-dev-server` 版本问题, 可以删除 `package.json` 中 `webpack-dev-server` 配置中的 `^`, 然后删除 `node_modules` 重新执行 `npm install`.
